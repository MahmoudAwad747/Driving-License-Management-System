# Driving & Vehicle License Department (DVLD) System

DVLD System is a **Driving License Management System**, designed to provide services for issuing, renewing, and replacing licenses, managing people and users, and organizing driving tests efficiently.

---

## 🌟 Main Features

The system focuses on **four main areas**:

### 1️⃣ Manage People
- Manage people data in the system  
- Add, edit, and delete person records  
- Easily search for people  

### 2️⃣ Users
- Manage system user accounts  
- Add, edit, and delete users  
- Control access permissions and login  

### 3️⃣ License Services
The system offers several license services:  

1. **New Local Driving License** – Issue a new local driving license  
2. **Renew Driving License** – Renew an existing license  
3. **Replacement for Damaged License** – Replace a damaged license  
4. **Replacement for Lost License** – Replace a lost license  
5. **Detain Licenses** – Detain a license  
6. **Release Detained License** – Release a detained license  
7. **Tests for Issuing the License** – Required driving tests:
   - **Vision Test**  
   - **Written Test**  
   - **Street Test**  
   - Users can **Retake Tests** if they fail  
   - Users can create **Test Appointments** to schedule tests  
8. **International License** – Issue an international license, available only for users who hold a (Local Driving License) **Class 3**

### 4️⃣ License Classes
License classification according to vehicle type:
- **Class 1** – Small Motorcycle  
- **Class 2** – Heavy Motorcycle License  
- **Class 3** – Ordinary Driving License  
- **Class 4** – Commercial  
- **Class 5** – Agricultural  
- **Class 6** – Small and Medium Bus  
- **Class 7** – Truck and Heavy Vehicle  

> Note: A license cannot be issued unless the user passes all required tests.

## 🖥️ Main Pages / Forms

- **Login Form** – User authentication

- **License Forms** – Issue, renew, and replace licenses
